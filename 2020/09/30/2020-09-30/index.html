What is Primitive Wrapper ?

Posted on 2020-09-30 In JS Knowledge

JavaScript data type

We all know that in JS, there are 2 kinds of data types, Primitive and object.

  • Primitive

    • string
    • number
    • null
    • undefined
    • symbol
    • boolean

  • Object

    • everything which is not Primitive type, and some constructors as below
    • String()
    • Number()
    • Boolean()
    • Array()
    • Object()
    • Function()
    • RegExp()
    • Date()
    • Error()
    • Symbol()
    • And so on…

For all constructors, it all can be used to build an object using new key words.
But, for some constructors, it looks similar with Primitive, such as

  • String()
  • Number()
  • Boolean()

But, don’t get confused, they are not Primitive type.
